# IntelliMail

Running Locally

1. Install Python and Ollama
2. `ollama serve`
3. `ollama use mistral` # Would be updating preferences based on modal performance.
4. `cd intelli-mail`
5. `pip install -r requirements.txt`
6. `python server.py`

Now your MCP server would be running on `http://localhost:8000/sse`
